# 📊 PhonePe Data Analytics Dashboard

## 📌 Project Overview

The PhonePe Data Analytics Dashboard is an interactive Business Intelligence project developed using Microsoft Power BI. The objective of this project is to analyze digital payment transactions and convert raw transactional data into meaningful business insights through interactive dashboards.

The dashboard provides an overview of transaction performance, user behavior, service usage, and transaction trends using KPI cards, charts, slicers, and report tooltips.

---

# 🎯 Project Objectives

* Analyze digital payment transaction data.
* Monitor overall business performance using KPIs.
* Identify transaction trends over time.
* Analyze service-wise transaction values.
* Understand user demographics.
* Build an interactive dashboard for business users.

# 📌 Business Insights

* Total Transaction Value reached ₹3.47 Billion.
* More than 300,000 transactions were analyzed.
* Overall Success Rate is 96%.
* Loan services generated the highest transaction value.
* Weekday transactions are higher than weekend transactions.
* The middle age group contributes the highest number of users.
